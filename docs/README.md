# Docs

- `windows-store-release.md`: end-to-end checklist for packaging and submitting the PWA to Microsoft Store.
- `windows-store-readiness-checklist.md`: practical done-versus-missing checklist for the current ParaBreach release.
- `store-listing-template.md`: reusable copy and asset checklist for the Store listing.
