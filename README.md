# ParaBreach

ParaBreach is a retro vector arcade defense game built as a lightweight static web app and installable Progressive Web App.

## Play Now

https://billbaud.github.io/ParaBreach/web/

## Controls

Move mouse, pen, or finger to aim.  
Hold click or tap to fire.

## Features

- Helicopter troop drops
- Missile defense
- Retro vector visuals
- Sound effects
- Installable PWA support

## Tech

- HTML
- Canvas
- JavaScript
- Service worker caching
- Web app manifest

## Local Development

1. Run `npm install`
2. Run `npm run dev`
3. Open `http://127.0.0.1:4173`

Use the VS Code launch profile to open the local dev server in Microsoft Edge once the server is running.
