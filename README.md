# ParaBreach

Retro vector arcade defence game.

## ▶ Play Now

https://billbaud.github.io/ParaBreach/web/

## Controls

Move mouse / finger to aim  
Hold click / tap to fire

## Features

- Helicopter troop drops
- Missile defence
- Retro vector graphics
- Sound effects
- Progressive Web App support
